API du Port de Plaisance Russell

Votre Mission

Le port de plaisance de Russell souhaite se doter d'une application web de gestion des réservations de catway (petit appontement pour amarrer un bateau). La capitainerie a mis en place cette API privée pour gérer :

  • Les catways et leurs caractéristiques
  • Les réservations des emplacements
  • Les utilisateurs de la capitainerie
